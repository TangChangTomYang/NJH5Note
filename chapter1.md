####十、兄弟选择器 

- 1、兄弟选择器有2个：**相邻兄弟选择器（CSS2 推出）、和通用兄弟选择器(CSS3推出)**.


- 2、**相邻兄弟选择器**<br>**作用:**给指选择器后面紧跟的那个选择器选中的标签设置属性.<br> **格式:**
```
目标选择器名称+紧跟选择器名称{
    属性名:属性值;
}
比如:
h1+p{
    color:red;
}
```
**注意点:**<br>(1)、相邻兄弟选择器必须使用`+` 链接.<br>(2)、相邻兄弟选择器中只能选中紧跟的那个标签,不能选中被隔开的标签.
![](/assets/Snip20180614_3.png)






- 3、**通用兄弟选择器**<br><br>**作用:**<br> 给指定选择器**后面的 所有的 选择器选中的 标签设置属性,不管是紧跟的还是隔开的**<br><br>**格式:**

    ```
    目标选择器名称~兄弟选择器名称{
            属性名:属性值;
    }
    比如:
    h1~p{
        color:red;
    }
    ```
    
    ![](/assets/Snip20180614_4.png)
    **注意:**<br>(1)、通用兄弟选择器必须使用`~` 链接<br>(2)、通选兄弟选择器选中的是指定选择器后面的某个选择器选中的所有标签,不管是否相连或者被隔开.

####十、序选择器 (选择第几个标签选择器)

- 1、**CSS3 中新增的选择器 最具代表性的就是序选择器**,主要有2种:**同级别第几个 和 同级别同类型第几个**<br>


<br><br>
- (1)、**同级别的第几个,取出同一个级别的第几个,如果是对应的标签就设置属性** <br><br>**格式:**
    ```
    标签名:同级别序号描述{
        属性名:属性值;
    }
    比如:
    p:first-child{
            color:red;
    }
    要表达的意思是,同级别中的第一个只要是p标签 就设置他的样式,否则就不满足要求.
    ```
    <br>
    - **:first-child,选中同级别中的第一个标签设置属性**
        ```
        p:first-child{
            color:red;
        }
        ```
    - **:last-child,选中同级别中的最后一个标签设置属性**
        
        ```
        p:last-child{
            color:red;
        }
        ```
    - **:nth-child,选中同级别中的第几个标签设置属性,n 可以自己任意定义**    
        ``` 
        p:nth-child(3){
            color:red;
        }
        ``` 
        ![](/assets/Snip20180614_7.png)<br>
    **注意点:**<br><br>**同级别中的第几个,不区分类型,只管取出同一级别的第几个然后判断是不是对应的标签,是就执行,不是就不执行. **
    <br><br>

- (2)、**同级别同类型的第几个,取出同一级别中同种类型的第几个,如果是对应标签就设置属性.**<br><br>**格式:**
```
标签名称:同级别同类型序号描述{
    属性名:属性值;
}
比如:
p:first-of-type{
    color:red;
}
```
    - **:first-of-type, 同级别 同类型的第一个**
    ```
    p:first-of-type{
        color:red;
    }
    ```
    - **:last-of-type, 同级别同类型的最后一个**
    ```
    p:last-of-type{
        color:red;
    }
    ```
    - **:nth-of-type(n), 同级别 同类型的第几个,n 可以自己设定**
    ```
    p:nth-of-type(3){
        color:red;
    }
    表示要取出同级别 同类型中的第三个标签, 如果是p 就设置属性,否则不设置.
    ```
![](/assets/Snip20180614_8.png)
**注意:**<br>**同级别同类型第几个 ,要区分类型和序号,也就是说,取出同级别中的同类型的第几个,如果是对应的标签就设置属性,否则不设置**
