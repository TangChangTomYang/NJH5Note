####什么是div?

- **div 的作用:**<br> **一般用于配合CSS 完成网页的布局**,网页一般分为3个部分,如下图: 

![](/assets/Snip20180703_1.png)

```
 .head{
    width: 980px;
    height: 100px;
    background: red;
    margin: auto;
}

.content{
    width: 980px;
    height: 500px;
    background: green;
    margin: auto;
}

.footer{
    width: 980px;
    height: 100px;
    background: orange;
    margin: auto;
}

<div class="head"></div>
<div class="content"></div>
<div class="footer"></div>

```
<br> <br>
####什么是span?

- **一般用于配合CSS 完成网页中一些局部的信息,如下:**

![](/assets/Snip20180703_3.png)
```
p{
    color: blue;
}

span{
    color: red;
}

<p>努力到 <span>无能为力</span> ,拼搏 <span>到感动自己</span> </p>
```
